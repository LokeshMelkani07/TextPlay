# TextPlay
This is  a Text Manipulating Website which can perform certain operations with the input text such as Convert Uppercase to Lowercase or Lowercase to Uppercase or copy the text , Count number of words and characters in input text and it has a Dark Mode also , Made by using HTML , CSS ,  React JS
